Información del Proyecto

El objetivo de este proyecto es crear una página web sobre el profesor Norma Cruz Balmaceda. El proyecto fue asignado por el profesor Candia.

Este sitio web incluye información detallada sobre el profesor, sus hobbies, intereses y experiencias educativas. Además, se incluyen secciones dedicadas a los creadores del sitio web.

El diseño está pensado para ser adaptable y accesible, asegurando que todos los usuarios puedan navegar fácilmente a través del contenido.

Códigos Ocupados y Su Implementación

A continuación, se describe el uso de los códigos y cómo implementarlos:

  • HTML: Se utiliza para estructurar las páginas. Todos los archivos HTML comparten elementos comunes como:
    • Encabezados (<'header'>)
    • Menús de navegación (<'nav'>)
    • Contenedores principales (<'div class="container"'>)
    • Pie de página (<'footer'>)
  • CSS: Se emplea para dar estilo a las páginas. Los estilos incluyen fuentes personalizadas, colores, animaciones (`fadeIn`, `slideIn`) y efectos visuales como sombras y bordes redondeados.
  • Animaciones CSS: Las animaciones se aplican mediante `@keyframes` para mejorar la experiencia del usuario. Ejemplos:
    • `fadeIn`: Hace que los elementos aparezcan gradualmente.
    • `slideIn`: Desliza los títulos desde arriba hacia abajo.
  • JavaScript: Se utiliza para controlar la funcionalidad del menú desplegable. Al hacer clic en el botón del menú, se alterna entre mostrar y ocultar el menú usando clases CSS (`visible` y `active`).
  • Imágenes y Multimedia: Se incluyen imágenes (<'img'>), audios (<'audio'>) y videos (<'video'>) para enriquecer el contenido.
  • Fondos Animados: Se utilizan GIFs como fondos para dar dinamismo visual. Ejemplo: `background-image: url('gif-proyecto.gif')`.

Estructura General Compartida

Todas las páginas comparten un encabezado, un contenedor principal y un pie de página. El menú incluye enlaces a las páginas principales: Inicio, Profesor, Creador y Proyecto.

Código de Cada Página

  • Inicio: Contiene botones interactivos para navegar a las páginas del profesor y creador.
  • Profesor: Incluye una foto del profesor, un reproductor de audio y detalles sobre sus hobbies.
  • Creadores: Presenta fotos, descripciones y un video de los creadores.
  • Proyecto: Explica el objetivo del proyecto y su diseño.

Animaciones

Se utilizaron animaciones como fadeIn y slideIn para mejorar la experiencia visual del usuario.

Recursos Utilizados

  • Imágenes: Logo (prepa.png), fotos del profesor y creadores.
  • Audios: Archivo de audio en la página del profesor (Musica-Norma.mp3).
  • Videos: Video de presentación en la página de los creadores (video-creador.mp4).
  • GIFs de Fondo: Fondos animados para las páginas específicas.
